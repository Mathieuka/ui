# Card

> OKP4 UI Card component.

[![stability-unstable](https://img.shields.io/badge/stability-unstable-yellow.svg)](https://github.com/emersion/stability-badges#unstable)

## Overview

Card is a component laid out on the page. A card can have a content, a header and a footer, all optional.
Content, header and footer accept React components to ensure maximum flexibility.
Any component picked from OPK4 UI library e.g. Typography gets styled with colors of the current theme setting.
