# Text component

[![stability-unstable](https://img.shields.io/badge/stability-unstable-yellow.svg)](https://github.com/emersion/stability-badges#unstable)

## Description

**Text** is the primitive component to display some text or pure HTML. It comes with limited styling capabilities to be easily integrated with OKP4 design system.

Any possible combination of the following propeties can be used and are detailed below.

## Usage

<Story
  name="Usage"

## Fonts

The text component can be styled with two font families:

### Main font

The main font is `Gotham`, it is also the default one.

<Story name="Main font" argTypes={{}} args={{}}>
  {args => (
    <Text as="div" fontFamily="brand">
      <p>ABCDEFGHIJKLMNOPQRSTUVWXYZ</p>
      <p>abcdefghijklmnopqrstuvwxyz</p>
      <p>123456789 !?.;:</p>
    </Text>
  )}
</Story>

### Secondary font

You can also choose to use a secondary font called `NotoSansMono`.

## Font weight

Depending on the need, the font weight can be modified. The values are `light`, `medium` (default), `bold`, `black`.
Internally, this parameter will actually select a variant of the selected font.
