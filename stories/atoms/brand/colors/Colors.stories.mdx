# Colors

> The colors palette defined at OKP4.

## Dark theme

This is the main theme used at OKP4 for all user interfaces, called "Blue Stone".  
This theme is rigorously modern, mystical and captivating, and symbolizes all the power and depth relating to our protocol.

## Light theme

This is the secondary theme used at OKP4 for all user interfaces, called "Light Stone".  
This theme was designed to provide the user with a clearer atmosphere, while maintaining the visual markers carried by the Blue Stone theme.
