## Themes

Themes make it possible to mark the use of OKP4 logos within an ensemble. The contrasts are important in order to ensure maximum integration in the interfaces.

### Dark

This is the reference theme of the logo, which makes it possible to magnify the use of the Blue Stone theme by using in particular a high contrast of white.

<Canvas>
  <Story
    name="Dark"
    args={{
      variant: 'logo',
      theme: 'dark',
      size: 'medium'
    }}
  >
    {args => (
      <div
        style={{
          background:
            'transparent linear-gradient(180deg, #0C1529 0%, #0F224A 100%) 0% 0% no-repeat padding-box',
          padding: '20px'
        }}
      >
        <Logo {...args} />
      </div>
    )}
  </Story>
</Canvas>
